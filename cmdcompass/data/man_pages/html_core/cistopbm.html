NAME

cistopbm - convert Compuserve RLE image to PBM

SYNOPSIS

cistopbm [-i] [cisfile]

DESCRIPTION

This program is part of Netpbm(1).

cistopbm reads a Compuserve RLE file as input. and produces a PBM image as output.

OPTIONS

In addition to the options common to all programs based on libnetpbm (most notably -quiet, see Common Options ), cistopbm recognizes the following command line option:

-i

Inverse: Reverse the mapping of foreground/background to black/white.

SEE ALSO

pbmtocis(1), pbm(1) CompuServe RLE file format

HISTORY

cistopbm was new in Netpbm 10.48 (September 2009).
